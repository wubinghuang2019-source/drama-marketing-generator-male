- **目标受众**：25-45岁男性观众
- **剧集类型**：谍战、刑侦、军旅、动作、反腐
- **风格**：硬核、专业、理性

- ✅ **微博热搜**（真实API接入，含热度值+分类标签）
- 抖音热榜（备用数据）
- B站热门（备用数据）
- 豆瓣热议（备用数据）

### 2. 2025营销框架
- **5层社交策略**：核心圈层-KOL-大众传播-长尾运营-数据反馈
- **4C内容模型**：Content/Community/Commerce/Conversion
- **TOFU-MOFU-BOFU转化漏斗**

### 3. 零成本运行
- 纯前端JavaScript实现
- 无后端API调用费用
- 微博热搜自动刷新（每5分钟）
